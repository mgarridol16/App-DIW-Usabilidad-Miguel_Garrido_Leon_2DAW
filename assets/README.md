# Assets Folder - Images Required

## Required Images

This folder needs two images for the AboutUs component to display correctly:

### 1. Profile Photo
- **Filename:** `profile-photo.jpg`
- **Description:** Your professional photo (the circular one from your CV)
- **Recommended size:** 400x400 pixels minimum
- **Format:** JPG or PNG
- **How to save:**
  1. Save your profile photo from the attachments
  2. Rename it to `profile-photo.jpg`
  3. Copy it to this `assets` folder

### 2. Europass Logo
- **Filename:** `europass-logo.png`
- **Description:** The official Europass logo
- **Format:** PNG (with transparency)
- **How to save:**
  1. Save the Europass logo from the attachments
  2. Rename it to `europass-logo.png`
  3. Copy it to this `assets` folder

## Fallback Behavior

If the images are not found:
- **Profile photo:** Will display a blue circle with your initials "MG"
- **Europass logo:** Will be hidden automatically
